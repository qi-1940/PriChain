# proj62-linux-lock-priority-inheritance

### 项目名称

Linux 锁优先级传递

### 项目描述

优先级翻转是指当一个高优先级任务通过锁等机制访问共享资源时, 该锁如果已被一低优先级任务占有, 将可能造成高优先级任务被许多具有较低优先级任务阻塞.
优先级翻转问题是影响 linux 实时性的障碍之一, 甚至引起很多稳定性问题.

### 特征

*	优先级继承、传递
*	实时性
